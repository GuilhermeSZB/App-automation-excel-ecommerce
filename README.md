# 📊 App Automation Excel E-commerce

Aplicação desenvolvida para automatizar processos de gerenciamento de dados em planilhas Excel voltadas para operações de e-commerce, proporcionando mais agilidade, organização e redução de erros manuais.

## 🚀 Funcionalidades

- 📥 Importação de dados de planilhas Excel
- 📊 Organização e tratamento de informações
- 🔄 Automação de tarefas repetitivas
- 📈 Geração de relatórios
- ✅ Redução de erros manuais
- ⚡ Processamento rápido de grandes volumes de dados

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- OpenPyXL
- Excel (.xlsx)
